## One-time setup (about 15 minutes)

### Step 1 — Upload files

1. Open [InfinityFree Control Panel](https://dash.infinityfree.com/) → your site → **Online File Manager** (or FTP).
2. Open folder **`htdocs`**.
3. Upload everything from the **`deploy`** ZIP (see below) **into `htdocs`**, not inside a subfolder.
4. After upload you should see `index.php`, `includes/`, `css/`, `assets/`, `config.php`, etc. directly in `htdocs`.

**ZIP location on your PC:** `deploy/FBAStransmittal-upload.zip` (generated by `deploy/package.ps1`)

### Step 2 — MySQL database

1. Control Panel → **MySQL Databases** → create database (note **hostname**, **username**, **password**, **database name**).
2. Open **phpMyAdmin** → select your database → **Import** → choose `sql/setup-hosting.sql` → **Go**.

### Step 3 — Edit `config.php` on the server

In File Manager, open `htdocs/config.php` and set:

| Key | Value |
|-----|--------|
| `db_host` | MySQL **hostname** from panel (e.g. `sql205.infinityfree.com`) — **not** your website address |
| `db_user` | MySQL username |
| `db_pass` | MySQL password |
| `db_name` | Full database name |
| `debug` | `true` while testing, then **`false`** |

Save the file.

### Step 4 — Test

Open: **https://fbastransmittal.infinityfree.io/**

- Create a test transmittal → **Generate & Save**
- Use **Search / Reprint** and print

If you see “Database connection failed”, fix `db_host` / user / password / `db_name` in `config.php`.

### Step 5 — Go live

- Set `'debug' => false` in `config.php`
- Delete `setup-logo.php` from the server if you do not need it
- Optional: add Hostinger domain (see [HOSTING.md](HOSTING.md))

---

## Update the site later

1. Run `deploy/package.ps1` on your PC (creates a fresh ZIP).
2. Upload changed files to `htdocs` (or replace all except `config.php`).
3. Do **not** overwrite `config.php` unless you intend to change database settings.
